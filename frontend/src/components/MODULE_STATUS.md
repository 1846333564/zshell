# 前端组件模块状态

## 范围

连接表单、监控面板、终端标签/面板和文件管理器等 Vue 组件。

## 重要文件

- `ConnectionForm.vue`
- `MonitorPanel.vue`
- `TerminalTabs.vue`
- `TerminalPanel.vue`
- `FileManager.vue`
- `RemoteCodeEditor.vue`

## 当前状态

终端和监控面板已经集成。`ConnectionForm.vue` 支持新建和编辑模式，并可选择前端模式、后端模式、运维模式。`TerminalPanel.vue` 处理终端聚焦字体缩放、字体大小持久化、终端剪贴板快捷键，以及复制、粘贴、清屏、重连的视口定位右键菜单。终端容器会抵消全局 UI zoom，避免 xterm.js 左键选择内容时选区相对鼠标位置左上偏移；终端 flex item 保持正常 100% 尺寸，不再把 UI 缩放值乘进宽高，避免右侧和下方留空。

`FileManager.vue` 提供顶部路径输入、路径历史弹层、全局已知目录 Tab 唯一补全、可折叠路径导航、固定 `/` 根路径可见、解析后的 home 路径如 `/root`、树节点 basename 展示、与右侧文件列表一致的目录树分割线和选中条、右侧居中折叠按钮、视口定位右键文件操作、点击外部关闭右键菜单、拖放上传、选择器上传、可折叠上传记录、后端真实 SFTP 写入进度、可调整文件列宽和多普通窗口在线文本编辑。它会按连接工作模式默认打开 `/var`、`/opt` 或 `/`，目录树和右侧列表共享同一个目录缓存；访问过的目录会先使用内存缓存秒开并同时刷新远端最新目录，已展开目录的一层子目录会按服务器 CPU 硬件线程数并发预加载，手动切换目录会取消旧预加载。编辑窗口可拖动、最小化、最大化，并支持多个文件同时打开；`RemoteCodeEditor.vue` 使用 Monaco Editor 替换原 `textarea`，支持代码高亮、Tab 缩进、`Ctrl+F` 搜索、匹配高亮和替换，并通过动态 import 与 worker 懒加载降低启动影响。文件复制/剪切剪贴板会校验 `copy`/`move` 动作，粘贴时使用当前剪贴板快照，避免复制操作沿用剪切状态；选中文件或文件夹后右键可强制删除，树上目录右键删除也会同步失效共享缓存。

## 已知工作

文件动作继续放在右键菜单中，不回退为固定工具栏按钮。真实服务器可用时，需要验证文件/目录右键、多编辑窗口、在线编辑保存和权限错误表现。
