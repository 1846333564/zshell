# 前端组件模块状态

## 范围

连接表单、监控面板、终端标签/面板和文件管理器等 Vue 组件。

## 重要文件

- `ConnectionForm.vue`
- `MonitorPanel.vue`
- `TerminalTabs.vue`
- `TerminalPanel.vue`
- `FileManager.vue`
- `useFileManager.js`
- `RemoteCodeEditor.vue`

## 当前状态

终端和监控面板已经集成。`MonitorPanel.vue` 现在 1 秒自动刷新，展示保存的服务器硬件信息、低成本实时系统信息、服务器时间、CPU/内存/磁盘、进程、分区和上下行网速折线图；网速图只显示峰值并带左侧纵轴刻度。`ConnectionForm.vue` 支持新建和编辑模式，并可选择前端模式、后端模式、运维模式。`TerminalPanel.vue` 处理终端聚焦字体缩放、字体大小持久化、终端剪贴板快捷键、动态主题刷新，以及复制、粘贴、清屏、重连的视口定位右键菜单。终端容器会抵消全局 UI zoom，避免 xterm.js 左键选择内容时选区相对鼠标位置左上偏移；终端 flex item 保持正常 100% 尺寸，不再把 UI 缩放值乘进宽高，避免右侧和下方留空。

`FileManager.vue` 保留模板和 props，文件管理业务 setup 放在 `useFileManager.js`。文件管理器提供顶部路径输入、路径历史弹层、全局已知目录 Tab 唯一补全、可折叠路径导航、固定 `/` 根路径可见、解析后的 home 路径如 `/root`、树节点 basename 展示、与右侧文件列表一致的目录树分割线和选中条、右侧居中折叠按钮、视口定位右键文件操作、点击外部关闭右键菜单、拖放上传、选择器上传、可折叠上传记录、后端真实 SFTP 写入进度、可调整文件列宽、列头排序点击动画、升降序箭头和多普通窗口在线文本编辑。它会按连接工作模式默认打开 `/var`、`/opt` 或 `/`，目录树和右侧列表共享同一个目录缓存；路径历史保存在内存中，按访问频次和最近访问排序，显示时让高优先级路径位于底部并默认滚到底部；访问过的目录会先使用内存缓存秒开并同时刷新远端最新目录，目录树索引按小批量写入，后台目录预加载会延后并只单并发读取当前目录少量直接子目录，手动刷新和 F5 会重新索引所有已打开目录，手动切换目录和打开文本编辑器都会取消旧预加载。编辑窗口可拖动、最小化、最大化，并支持多个文件同时打开；`RemoteCodeEditor.vue` 使用 Monaco Editor 替换原 `textarea`，支持代码高亮、Tab 缩进、`Ctrl+F` 搜索、匹配高亮和替换，并通过动态 import 与 worker 懒加载降低启动影响；打开远程文本文件时会在编辑窗口中显示真实下载字节进度、速度和读取阶段，远程内容读完后才挂载 Monaco，超过限制、后端返回读取错误或流式读取未返回文件时会明确显示错误而不是空白文件；编辑器进度、窗口激活、关闭提示和上传禁用状态的模板绑定已显式接入，避免读取阶段或窗口交互触发 Vue 渲染异常。文件复制/剪切剪贴板会校验 `copy`/`move` 动作，粘贴时使用当前剪贴板快照，避免复制操作沿用剪切状态；选中文件或文件夹后右键可强制删除，树上目录右键删除也会同步失效共享缓存。

## 已知工作

文件动作继续放在右键菜单中，不回退为固定工具栏按钮。真实服务器可用时，需要验证文件/目录右键、多编辑窗口、在线编辑保存和权限错误表现。
