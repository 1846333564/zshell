## 当前状态

终端和监控面板已经集成。`MonitorPanel.vue` 现在 1 秒自动刷新，展示保存的服务器硬件信息、低成本实时系统信息、服务器时间、CPU/内存/磁盘、进程、分区和上下行网速折线图；网速图只显示峰值并带左侧纵轴刻度。监控刷新期间继续显示上一帧服务器时间，不再切换成“更新中”，成功响应后在时间末尾显示请求延迟毫秒数。`ConnectionForm.vue` 支持新建和编辑模式，并可选择前端模式、后端模式、运维模式。`TerminalPanel.vue` 处理终端聚焦字体缩放、字体大小持久化、终端剪贴板快捷键、动态主题刷新，以及复制、粘贴、清屏、重连的视口定位右键菜单。终端容器会抵消全局 UI zoom，避免 xterm.js 左键选择内容时选区相对鼠标位置左上偏移；终端 flex item 保持正常 100% 尺寸，不再把 UI 缩放值乘进宽高，避免右侧和下方留空。

`FileManager.vue` 保留模板和 props，文件管理业务 setup 放在 `useFileManager.js`，路径、格式化和常量工具拆在 `fileManagerUtils.js`，目录树、右键目标、虚拟区间和路径改写纯逻辑拆在 `fileTreeModel.js` 并配有 Node 测试，地址栏的父目录索引、二分前缀查询和交互状态拆在 `pathCompletion.js` / `usePathCompletion.js`。`ChevronIcon.vue` 提供统一的 90 度空心折线 SVG，目录树按右/下方向表示折叠状态，文件表头按上/下方向表示排序。文件管理器提供顶部路径输入、路径历史弹层、复用递归预加载缓存的多级 Tab/Enter 联动补全、可折叠路径导航、固定 `/` 与 home 锚点、仅目录左树、每个目录独立一行、左右虚拟滚动、整行选中并展开/折叠、目录/文件分离右键菜单、空白父目录菜单、原位重命名、拖放/选择器上传、整目录下载与复制/剪切、可折叠上传记录、后端真实 SFTP 写入进度、可调整文件列宽、列头排序点击动画和多普通窗口在线文本编辑。补全提示只在实际输入且存在匹配时出现，每次最多渲染 10 个当前父目录的直接子目录；Tab 按唯一目录、公共前缀、精确目录与唯一更长候选规则补全并同步导航左右区域，Enter 优先确认精确目录；补全导航期间会忽略输入框因 loading 禁用而触发的重复 `change`，避免旧草稿路径发起第二次导航。刷新目录时保留用户选择的折叠状态。右键菜单已移除顶部路径标题，并会在渲染后按真实高度重新定位到视口内。树节点右键遵循“已选中取自身、未选中取父目录”，右侧项目右键始终取鼠标下项目，空白或表头取当前目录；右侧项目点击会把左树定位到对应父目录。复制、剪切、上传和重命名后会定向刷新目标及同连接源父目录，防止非当前分支保留旧节点。它会按连接工作模式默认打开 `/var`、`/opt` 或 `/`，目录树和右侧列表共享同一个目录缓存；路径历史保存在内存中，按访问频次和最近访问排序，显示时让高优先级路径位于底部并默认滚到底部；访问过的目录会先使用内存缓存秒开并同时刷新远端最新目录，后台目录预加载会在当前目录加载完成后立即启动，每批最多挑 10 个未缓存目录，并基于已缓存目录继续递归推进，手动刷新和 F5 会重新索引所有已打开目录，手动切换目录和打开文本编辑器都会取消旧预加载。编辑窗口可拖动、最小化、最大化，并支持多个文件同时打开；`RemoteCodeEditor.vue` 使用 Monaco Editor 提供完成下载后的代码高亮、Tab 缩进、`Ctrl+F` 搜索、匹配高亮和替换，并通过动态 import 与 worker 懒加载降低启动影响；打开远程文本文件时窗口和下载进度会立即显示，XHR 在独立网络事件中持续更新真实字节进度，并把已到达文本原地追加到独立只读预览，完整响应到达后才把全文交给 Monaco、完成布局和语言切换并开放编辑保存。真正关闭窗口会中止对应请求，超过限制或后端读取错误会明确显示；编辑器进度、窗口激活、关闭提示和上传禁用状态的模板绑定已显式接入。

在线编辑下载路径已从前端主线程的 `ReadableStream + NDJSON + Base64` 改为原始 UTF-8 XHR。XHR 同时上报真实字节进度并按字符偏移增量回调 `responseText`；`useFileManager.js` 在非响应式 pending 数组中收集文本，按约 1 秒向 raw `contentChunks` 发布合并批次并递增版本，完成时同步发布尾块。`RemoteCodeEditor.vue` 使用同步 watcher 把每个批次作为独立文本节点追加到只读预览，不再通过 `setRangeText` 反复重建累计全文，也不依赖下载期间的 Monaco 重绘。完成后再统一设置 `content` / `originalContent`、同步 Monaco 全文并原子切换显示。每个窗口的读取控制器放在非响应式 Map 中，直接关闭、保存并关闭、放弃关闭和组件卸载都汇聚到取消逻辑，取消后的回调会被请求身份检查忽略。

## 已知工作

文件动作继续放在右键菜单中，不回退为固定工具栏按钮。`useFileManager.js` 已先拆出纯工具模块，仍需要继续按目录浏览、预加载、编辑器窗口和文件动作拆分。真实服务器可用时，需要验证文件/目录右键、多编辑窗口、在线编辑保存、递归目录预加载和权限错误表现。
