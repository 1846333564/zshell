# 前端模块状态

## 范围

Wails 桌面窗口中的 Vue/Vite 前端。

## 重要文件

- `src/App.vue`：应用模板壳。
- `src/composables/app`：连接标签、更新弹窗、UI 缩放和窗口控制组合函数。
- `src/style.css` 与 `src/styles`：全局样式入口和分包样式。
- `src/components`
- `src/services`

## 当前状态

UI 是双栏桌面布局，包含自定义无边框窗口顶栏、左侧监控面板，以及右侧连接标签、终端和文件区域。未连接首页会隐藏监控侧栏，只保留已保存连接和连接配置区域；连接失败信息会在配置面板内换行，不再撑开首页布局。保存连接管理使用后端配置 API，不把 SSH 密码保存在浏览器本地存储，并支持前端、后端、运维三种工作模式；应用启动后短暂等待就绪并在配置加载或连接进行中禁用连接动作，避免刚打开窗口时快速点击造成重复连接或渲染高负载。应用壳会屏蔽非文件区系统右键菜单，同时允许文件管理器自定义菜单；非终端 UI 缩放、终端字体大小、主题键和自定义主题颜色都通过后端偏好持久化；默认主题保持原有 zShell 配色，`UI管理 -> 主题设置` 可切换 Dracula、Nord、Tokyo Night、Catppuccin Mocha、Gruvbox Dark、One Dark、Solarized Dark 或自定义颜色，并同步影响应用外壳、xterm 终端和 Monaco 编辑器；终端区域会抵消全局 UI zoom，避免 xterm.js 鼠标选择坐标相对光标左上偏移，并保持正常 100% flex 尺寸，避免缩放二次参与布局造成右侧或下方空白；文件和终端右键菜单渲染在视口层，避免缩放导致菜单坐标偏移；全局滚动条使用应用风格。Monaco Editor 会在应用启动后延后到更空闲的阶段预热，并在编辑器窗口打开时按需加载，不进入首屏静态 bundle。

`src/App.vue` 现在提供“关于 zShell”弹窗和“检查更新”流程；应用启动后会后台静默检查更新，只有发现新版本时才弹窗；更新弹窗在发现新版本或遇到更新错误时会保留 GitHub Release 下载页入口，确认更新后显示阶段进度、下载字节进度、重试和校验日志，并在下载、校验等可取消阶段提供“停止”按钮。左侧监控面板 1 秒自动刷新，显示保存的硬件信息、低成本实时系统信息、服务器时间和上下行网速折线图，网速图只显示峰值并带左侧纵轴刻度。文件区域包含 Monaco 在线文本编辑，编辑器以多个普通浮动窗口显示，支持代码高亮、Tab 缩进、`Ctrl+F` 搜索、替换、拖动、最小化、最大化、保存和脏关闭处理，并在打开远程文本文件时显示真实下载字节进度、速度和读取阶段；远程内容读完后会先以基础文本区显示并可编辑，Monaco 加载完成后再切换到完整编辑器。连接成功后会使用后端返回的服务器硬件信息。文件管理器按连接工作模式默认进入 `/var`、`/opt` 或 `/`，访问过的目录会先用内存缓存渲染，再刷新真实远端目录，并提供按频次/最近访问排序且默认滚到底部的路径历史、全局已知目录 Tab 唯一补全、低并发分批目录预加载、分批目录树索引、手动刷新重索引所有已打开目录、固定面包屑/表头、列头排序动画、可折叠上传记录和真实远端上传进度。

前端大文件已按职责拆分：`App.vue` 的业务状态移入 `src/composables/app`，`FileManager.vue` 的 setup 移入 `src/components/useFileManager.js`，全局样式拆入 `src/styles`。当前 `frontend/src` 下 Vue、JS 和 CSS 业务文件均低于 500 行。
