# Frontend Module Status

## Scope

Vue/Vite frontend for the Wails desktop window.

## Important Files

- `src/App.vue`
- `src/style.css`
- `src/components`
- `src/services`

## Current State

The UI is a two-column desktop layout with left monitor panel and right connection tabs, terminal, and file area. Saved connection management now uses backend config APIs instead of browser-local password history.

## Known Work

Run visual smoke checks against the packaged Wails window after large layout changes.
