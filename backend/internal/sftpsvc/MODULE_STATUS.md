# SFTP 服务模块状态

## 范围

远程目录列表、上传、下载、文本文件读写、zip 归档生成、同服务器快速文件动作和服务器间复制/移动。

## 重要文件

- `service.go`
- `service_test.go`

## 当前状态

支持目录列表、多文件/多目录上传、直接下载、面向在线编辑的 10 MB 远程文本读写、递归目录归档下载、跨服务器 SFTP 流式复制/移动，以及同服务器远端 `cp`/`mv` 快速复制/移动和远端 `rm -rf` 快速删除。目录列表、上传、在线读写和传输会复用共享 SSH 客户端，减少重复握手成本；上传可按 SFTP 写入字节回调进度，供 HTTP NDJSON 流式返回。复制粘贴会为同目录或同名冲突生成唯一目标名，例如 `name copy.ext`，并阻止目录复制/移动到自身或子目录；删除会拒绝空路径、根路径和不明确的相对路径。目录列表会用 SFTP `RealPath` 解析请求路径，返回解析后的路径，包含 mode 和 UID:GID owner 元数据，并按文件夹优先排序。

## 已知工作

后端路径 API 需要继续同时支持 `~` 和 `/`。深层目录、大文件传输、权限拒绝路径、文本保存权限、远端 `cp`/`mv`/`rm` 可用性和是否解析友好用户名，都需要真实服务器验证。
